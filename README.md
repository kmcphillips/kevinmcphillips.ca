# kevinmcphillips.ca

This is a simple hompage site, for myself.


## Getting started

```
git clone git@github.com:kmcphillips/kevinmcphillips.ca.git
cd kevinmcphillips.ca
bundle install
bundle exec rackup
```

Then open:

[http://localhost:9292/](http://localhost:9292/)


## Deploy

Using [fly.io](https://fly.io/) `fly deploy`
